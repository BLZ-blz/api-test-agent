# 后端接口自动化测试与文档生成Agent

## 📌 项目简介
这是一个基于多Agent协作的后端接口自动化测试与文档生成工具，旨在解决课程项目开发中接口文档与代码实现不同步、手动编写测试用例效率低、回归测试覆盖不全的痛点，提升团队开发协作效率。

## ✨ 核心功能
1.  **接口解析Agent**：自动读取项目接口定义（如SpringBoot Controller注解、OpenAPI规范），解析接口路径、参数、请求方式与响应结构，完成结构化梳理。
2.  **用例生成Agent**：基于解析结果进行长链推理，生成覆盖正常场景、异常参数、边界值、权限校验的多维度测试用例。
3.  **执行验证Agent**：调用轻量测试框架自动运行接口测试，采集响应结果、断言校验与错误日志，对失败用例进行归因分析。
4.  **文档同步Agent**：根据测试结果自动更新Markdown格式接口文档与测试报告，形成“解析-生成-执行-更新”的自动化闭环。

## 🚀 使用流程
1.  配置项目路径与接口信息
2.  运行Agent启动脚本
3.  查看生成的测试报告与接口文档

## 📊 项目成果
- 已在课程项目中落地使用，覆盖全部32个后端接口
- 将接口测试与文档维护的人工耗时从4小时/周压缩至15分钟/周，效率提升超90%
- 项目接口缺陷率降低85%，减少了因文档不一致导致的沟通成本
